# Open Uploader

A cross-platform desktop application for managing and uploading files to multiple cloud storage providers.

## ‚ú® Features

### Multi-Provider Support

- **Amazon S3** - Full support for AWS S3 buckets
- **Alibaba Cloud OSS** - Integrated Aliyun OSS support
- **Tencent Cloud COS** - Complete Tencent Cloud COS integration
- **Supabase Storage** - Native Supabase storage support
- **Custom S3-Compatible** - Support for any S3-compatible storage providers (MinIO, Cloudflare R2, etc.)

### File Management

- **Upload Files** - Easy upload to cloud storage buckets
- **Browse & Manage** - Browse and manage files across different providers
- **Batch Operations** - Perform batch file operations efficiently
- **Upload History** - Track all your upload history
- **Image Preview** - Preview images with blurhash placeholders
- **Drag & Drop** - Drag-and-drop file upload support

### Image Compression

- **Built-in Presets** - Multiple compression presets (Cover, Card, Thumbnail, Content, Original)
- **Custom Presets** - Create custom compression presets with configurable parameters
- **Format Conversion** - Support WebP, JPEG, and PNG output formats
- **Quality Control** - Adjustable quality, dimensions, and compression settings
- **Fit Modes** - Multiple fit modes (cover, contain, fill, inside, outside)
- **Aspect Ratio** - Control aspect ratio (16:9, 4:3, 1:1, free)
- **Live Preview** - Real-time compression preview with before/after comparison
- **Image Cropping** - Built-in cropping tool with aspect ratio lock
- **Batch Compression** - Compress multiple images at once
- **Statistics** - View compression statistics and file size reduction percentage

### Provider Management

- **Multiple Providers** - Add and configure multiple storage providers
- **Connection Testing** - Test connections before saving
- **Bucket Management** - Manage multiple buckets for each provider
- **Quick Access** - Sidebar for quick access to recently used providers

### User Interface

- **Dark Mode** - Full dark mode support
- **Responsive Layout** - Responsive design with collapsible sidebar
- **Recent Buckets** - Quick navigation to recently used buckets
- **Active Route Indicators** - Clear visual indication of active routes
- **Type-Safe Navigation** - Built with TypeScript for type-safe navigation

## üõ† Tech Stack

- **Electron** - Cross-platform desktop application framework
- **React 19** - UI framework with TypeScript
- **Vite** - Fast build tool and development server
- **TanStack Router** - Type-safe file-based routing
- **TanStack Query** - Data fetching and caching
- **tRPC** - End-to-end type-safe API layer
- **Drizzle ORM** - TypeScript ORM with PGlite database
- **Tailwind CSS v4** - Utility-first CSS framework
- **shadcn/ui** - Radix UI component library

- **Zustand** - State management
